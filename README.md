# 🏠 Home Assistant Glassmorphism Dashboard

A breathtaking, modern UI template for Home Assistant dashboards, built with HTML, CSS, and JS using the Glassmorphism design trend.

## ✨ Features
- **Stunning Aesthetics:** Blurred backgrounds, glowing animated orbs, and translucent glass cards.
- **Interactive UI:** Click on entities to toggle their states with smooth CSS transitions.
- **Lightweight:** Pure Vanilla HTML/CSS/JS. Easy to integrate into existing Home Assistant custom cards (like `custom:button-card` or `markdown` cards).
- **Responsive:** Scales perfectly across tablets, phones, and desktop screens.

## 🚀 How to Use
1. Clone this repository.
2. Open `index.html` in any modern browser to preview the UI.
3. Adapt the CSS classes into your Home Assistant Lovelace configuration using `lovelace-layout-card` or custom HTML cards.

## 🎨 Design System
- **Font:** Inter (Google Fonts)
- **Colors:** Deep Slate `#0f172a`, Glass `#ffffff1a`, Active `#4ade80`
